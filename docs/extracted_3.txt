ИСТОЧНИКИ ИНФОРМАЦИИ ДЛЯ РАСЧЕТОВ:
Все данные о пациентах с ИСМП («числители» и часть «знаменателей») - по результатам эпидемиологического расследования каждого случая инфекции + рутинно обязательно собираемые данные.
